Project Description

This is the first and only individual (as opposed to team) project this semester.

Term: Fall 2021

  • Project title: Words, Words, Words

  • This project is conducted by Emily Jennings-Dobbs

  • Project summary: The way philosophers talk and write has confused countless people for generations, myself included. Well, if you cant beat them, join them. Have you ever wanted to confuse people the way Aristotle and Plato have for generations? Well now you can! With this analysis well try to dissect the essence of word choice in philosophical text and really determine how to speak like a Philosophical writer.
